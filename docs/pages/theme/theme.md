# About this theme

>  本主题基于[VitePress](https://vitepress.dev/)默认主题修改

## 安装 <Badge type="info" text="Doing" />

详细可参考官方文档：https://vitepress.dev/guide/getting-started


```sh
mkdir vitepress
cd vitepress
npm install -D vitepress
npx vitepress init
```

运行：

```sh
npm run docs:dev
```
